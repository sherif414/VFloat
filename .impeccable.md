## Design Context

### Users

Developers, framework authors, and designers building complex UI components that require precise positioning (like tooltips, popovers, and dropdowns). They are highly technical and appreciate performance, accuracy, and clear visual demonstrations of logic.

### Brand Personality

**Precise, Architectural, High-Performance.**
VFloat is a tool of mathematics and layout logic. It should evoke the feeling of a high-end drafting table, a technical blueprint, or a 3D modeling environment.

### Aesthetic Direction: The Spatial Blueprint

- **Visual Tone:** Minimal Brutal Elegance. Sharp lines, zero border-radius elements, high contrast.
- **Color Palette:** Deep graphite/charcoal base (`#111317` surface) accented by high-energy "Electric Cyan" or "Laser Teal" (`#00E5FF`). No muddy greys or pure black.
- **Typography:** Architectural, geometric sans-serif (e.g., Space Grotesk) mixed with a highly legible monospace for data points/coordinates.
- **Theme:** Dark Mode native. It should feel like a backlit technical display.
- **Anti-References:** No "bubbly" SaaS templates, no thick soft shadows, no generic tech-blue gradients.

### Design Principles

1. **The \"No-Line\" Rule:** Use tonal shifts (lightness changes) and precise negative space rather than explicit 1px strokes to define boundaries and hierarchy.
2. **Intentional Asymmetry:** The layout should avoid boring centralization and instead use heavy left-justification balanced by negative space on the right, mimicking editorial spreads.
3. **Tonal Stacking over Drop Shadows:** Depth is achieved by lightening surfaces as they stack \"higher\" (closer to the user), utilizing glassmorphism only for absolutely floating elements.
4. **Data as Decoration:** Expose the "math" behind the UI. Use tiny crosshairs, subtle grid motifs, and coordinate markers (`[x: 0, y: 15]`) as decorative UI accents.
