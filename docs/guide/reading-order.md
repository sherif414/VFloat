# Reading Order

If you are new to VFloat, this is the path that gives the least friction:

1. [Start Here](/guide/) to get the quick orientation and first working example.
2. [Philosophy](/guide/philosophy) to understand the product boundaries and doc structure.
3. [Core Concepts](/guide/concepts) to learn the anchor, floating element, and context model.
4. [Interactions](/guide/interactions) to learn how open state changes in real components.
5. [Middleware](/guide/middleware) to tune placement, collision handling, and size.
6. [Virtual Elements](/guide/virtual-elements), [Keyboard List Navigation](/guide/list-navigation), and [Safe Polygon](/guide/safe-polygon) for focused advanced workflows.

## Jump Ahead If You Need To

- Building a tooltip or popover: start with [Interactions](/guide/interactions) and [Middleware](/guide/middleware).
- Building a menu or listbox: go to [Keyboard List Navigation](/guide/list-navigation).
- Following pointer coordinates or handling context menus: go to [Virtual Elements](/guide/virtual-elements).
- Smoothing hover transitions between trigger and panel: go to [Safe Polygon](/guide/safe-polygon).
