# Philosophy

VFloat is a Vue 3 positioning library. It helps you decide where a floating surface should appear and how it should respond to user input, without forcing you into a component system or a design system.

## What VFloat Is

- A positioning core built around `useFloating`.
- A set of interaction composables for click, hover, focus, dismissal, and keyboard navigation.
- A middleware pipeline for tuning coordinates, collision handling, and sizing.
- A small, composable surface that fits into your own Vue components.

## What VFloat Is Not

- Not a component library with prebuilt tooltips, menus, or dialogs.
- Not a styling system.
- Not a fork of Floating UI.
- Not a tooltip-only package.

## How The Docs Work

- Use the guide pages when you want a mental model, a workflow, or a recipe.

## Working With State

The context setter gives you a single place to coordinate visibility changes across click, hover, focus, and dismissal. Keep the guide focused on the workflow and use the API page for the exact call shape.
